# 008 フロントエンド - 共通コンポーネント

## 概要
複数画面で使用する共通UIコンポーネント（削除確認ダイアログ、回答修正モーダル）を実装する。

### 回答登録モーダル（`src/components/AnswerFormModal.tsx`）
- [ ] props：`isOpen`（boolean）・`questionId`（number）・`onSuccess`（() => void）・`onCancel`（() => void）
- [ ] フィールド：回答内容（テキストエリア、500文字以内）・回答者（テキスト、50文字以内）
- [ ] モーダル表示時（`isOpen` が `true` になったとき）に入力値・バリデーションエラー・APIエラーを初期化する
- [ ] 「登録」ボタン押下時：フロントバリデーション → `createAnswer`（`src/api/answers.ts`）を呼ぶ → 成功時 `onSuccess` を呼ぶ
  - `createAnswer` は `POST /api/questions/{id}/answers` を呼び出す
- [ ] 「キャンセル」ボタン押下時：`onCancel` を呼ぶ
- [ ] バリデーションエラーを各フィールド直下に表示する
- [ ] APIエラーのうち `fieldErrors` が存在する場合：各フィールド直下に表示する
- [ ] APIエラーのうち `fieldErrors` が存在しない場合：フォーム上部に一般エラーとして表示する
- [ ] 成功時は `onSuccess()` のみ呼ぶ。モーダルクローズ・一覧再取得は親コンポーネントで制御する
