### 共通 API クライアント（`src/api/client.ts`）

- [ ] `ApiError` クラスを定義する
  - `status`, `error`, `message`, `timestamp`, `fieldErrors?` を保持する
  - `new ApiError(response: ErrorResponse)` で生成できるようにする
  ```ts
  class ApiError extends Error {
    status: number;
    error: string;
    timestamp: string;
    fieldErrors?: FieldError[];

    constructor(res: ErrorResponse) {
      super(res.message);
      this.status = res.status;
      this.error = res.error;
      this.timestamp = res.timestamp;
      this.fieldErrors = res.fieldErrors;
    }
  }
  ```

- [ ] `BASE_URL` を `VITE_API_BASE_URL` 環境変数から読み込む定数として定義する
  - 未設定時は `http://localhost:8080` をデフォルト値とする
  ```ts
  const BASE_URL = import.meta.env.VITE_API_BASE_URL ?? 'http://localhost:8080';
  ```

- [ ] 共通 fetch 関数 `apiFetch` を実装する
  - 引数：`path: string`, `init?: RequestInit`
  - `Content-Type: application/json` をデフォルトヘッダーに付与する
  - レスポンスが 4xx / 5xx の場合は JSON をパースして `ApiError` としてスローする
  - 成功時はレスポンス JSON をそのまま返す（型パラメータ `<T>` で呼び出し側が指定）
  ```ts
  async function apiFetch<T>(path: string, init?: RequestInit): Promise<T> {
    const res = await fetch(`${BASE_URL}${path}`, {
      ...init,
      headers: { 'Content-Type': 'application/json', ...init?.headers },
    });
    if (!res.ok) {
      const errBody: ErrorResponse = await res.json();
      throw new ApiError(errBody);
    }
    return res.json() as Promise<T>;
  }
  ```

- [ ] `apiFetch` と `ApiError` を export する
