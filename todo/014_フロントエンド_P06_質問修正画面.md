# 014 フロントエンド - P-06 質問修正画面

## 概要
既存質問の内容を編集・更新する画面を実装する。

## 画面パス
`/questions/:id/edit` → `QuestionEditPage.tsx`

## Todo

### UI実装（`src/pages/QuestionEditPage.tsx`）
- [ ] タイトル入力欄（テキスト、100文字以内）に既存値を初期表示する
- [ ] 質問内容入力欄（テキストエリア、500文字以内）に既存値を初期表示する
- [ ] 質問者入力欄（テキスト、50文字以内）に既存値を初期表示する
- [ ] 「更新」ボタンを配置する
- [ ] 「戻る」ボタンを配置する → クリックで `/` へ遷移する（`history.back` は使用しない）

### API連携
- [ ] 画面初期表示時に `GET /api/questions/{id}` を呼び、既存データをフォームに展開する
- [ ] 対象が存在しない場合（404）：「対象データが見つかりません。」を表示する
- [ ] 「更新」ボタン押下時：フロントバリデーション → `PUT /api/questions/{id}` を呼ぶ
  - リクエストボディに `status` は含めない
- [ ] 更新成功時：`/questions/:id` へリダイレクトする
- [ ] APIバリデーションエラー（400 + `fieldErrors`）：各フィールドにエラーを表示する
- [ ] APIエラー（500）：「システムエラーが発生しました。しばらくしてから再度お試しください。」を表示する

### バリデーション
- [ ] タイトル：必須（「タイトルは必須です。」）・100文字以内（「タイトルは100文字以内で入力してください。」）
- [ ] 質問内容：必須（「質問内容は必須です。」）・500文字以内（「質問内容は500文字以内で入力してください。」）
- [ ] 質問者：必須（「質問者は必須です。」）・50文字以内（「質問者は50文字以内で入力してください。」）
- [ ] エラーは各フィールド直下にインライン表示する
