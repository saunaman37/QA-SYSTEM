# 002 DB設計・テーブル作成

## 概要
`questions` テーブルと `answers` テーブルを定義するエンティティを実装し、JPA によってテーブルが自動生成されることを確認する。

## Todo

### Question エンティティ（`entity/Question.java`）
- [⚪︎] `@Entity` `@Table(name = "questions", indexes = {...})` を付与する（インデックスは後述）
- [⚪︎] `id`：`@Id` `@GeneratedValue(strategy = GenerationType.IDENTITY)`（BIGINT AUTO_INCREMENT）
- [⚪︎] `title`：`@Column(nullable = false, length = 100)`（VARCHAR(100) NOT NULL）
- [⚪︎] `content`：`@Lob` `@Column(nullable = false)`（TEXT NOT NULL）
- [⚪︎] `questioner`：`@Column(nullable = false, length = 50)`（VARCHAR(50) NOT NULL）
- [⚪︎] `status`：`@Enumerated(EnumType.STRING)` `@Column(nullable = false)`、型は `QuestionStatus` enum、デフォルト `UNANSWERED`
  ```java
  public enum QuestionStatus { UNANSWERED, ANSWERED }
  ```
- [⚪︎] `createdAt`：`@Column(name = "created_at", nullable = false)`（DATETIME NOT NULL）
- [⚪︎] `updatedAt`：`@Column(name = "updated_at", nullable = false)`（DATETIME NOT NULL）
- [⚪︎] `deletedAt`：`@Column(name = "deleted_at")`（DATETIME NULL）
- [⚪︎] `@PrePersist` で `createdAt` / `updatedAt` に `LocalDateTime.now(ZoneId.of("Asia/Tokyo"))` を設定する
- [⚪︎] `@PreUpdate` で `updatedAt` に `LocalDateTime.now(ZoneId.of("Asia/Tokyo"))` を設定する
- [⚪︎] `answers` との OneToMany 関連を定義する（`mappedBy = "question"`）

### Answer エンティティ（`entity/Answer.java`）
- [⚪︎] `@Entity` `@Table(name = "answers", indexes = {...})` を付与する（インデックスは後述）
- [⚪︎] `id`：`@Id` `@GeneratedValue(strategy = GenerationType.IDENTITY)`（BIGINT AUTO_INCREMENT）
- [⚪︎] `question`：`@ManyToOne` `@JoinColumn(name = "question_id", nullable = false)`（FK → questions.id）
- [⚪︎] `content`：`@Lob` `@Column(nullable = false)`（TEXT NOT NULL）
- [⚪︎] `responder`：`@Column(nullable = false, length = 50)`（VARCHAR(50) NOT NULL）
- [⚪︎] `createdAt`：`@Column(name = "created_at", nullable = false)`（DATETIME NOT NULL）
- [⚪︎] `updatedAt`：`@Column(name = "updated_at", nullable = false)`（DATETIME NOT NULL）
- [⚪︎] `deletedAt`：`@Column(name = "deleted_at")`（DATETIME NULL）
- [⚪︎] `@PrePersist` で `createdAt` / `updatedAt` に `LocalDateTime.now(ZoneId.of("Asia/Tokyo"))` を設定する
- [⚪︎] `@PreUpdate` で `updatedAt` に `LocalDateTime.now(ZoneId.of("Asia/Tokyo"))` を設定する

### インデックス
- [⚪︎] `questions` テーブルに `@Table` の `indexes` 属性で以下を追加する
  - `status`（`@Index(name = "idx_questions_status", columnList = "status")`）
  - `created_at`（`@Index(name = "idx_questions_created_at", columnList = "created_at")`）
  - `deleted_at`（`@Index(name = "idx_questions_deleted_at", columnList = "deleted_at")`）
- [⚪︎] `answers` テーブルに `@Table` の `indexes` 属性で以下を追加する
  - `question_id`（`@Index(name = "idx_answers_question_id", columnList = "question_id")`）
  - `deleted_at`（`@Index(name = "idx_answers_deleted_at", columnList = "deleted_at")`）

### リポジトリ
- [⚪︎] `QuestionRepository.java`：`JpaRepository<Question, Long>` を継承する
- [⚪︎] `AnswerRepository.java`：`JpaRepository<Answer, Long>` を継承する

### 動作確認
- [ ] アプリ起動時に `questions` / `answers` テーブルが自動生成されることを確認する（`ddl-auto: update`）
- [ ] タイムスタンプが Asia/Tokyo で保存されることを確認する
