# 002 DB設計・テーブル作成

## 概要
`questions` テーブルと `answers` テーブルを定義するエンティティを実装し、JPA によってテーブルが自動生成されることを確認する。

## Todo

### Question エンティティ（`entity/Question.java`）
- [ ] `@Entity` `@Table(name = "questions")` を付与する
- [ ] `id`：`@Id` `@GeneratedValue` (BIGINT, AUTO_INCREMENT)
- [ ] `title`：VARCHAR(100) NOT NULL
- [ ] `content`：TEXT NOT NULL（`@Column(columnDefinition = "TEXT")`）
- [ ] `questioner`：VARCHAR(50) NOT NULL
- [ ] `status`：ENUM(`UNANSWERED`, `ANSWERED`)、デフォルト `UNANSWERED`
- [ ] `createdAt`：DATETIME NOT NULL（`@Column(name = "created_at")`）
- [ ] `updatedAt`：DATETIME NOT NULL（`@Column(name = "updated_at")`）
- [ ] `deletedAt`：DATETIME NULL（`@Column(name = "deleted_at")`）
- [ ] `@PrePersist` で `createdAt` / `updatedAt` に現在時刻（Asia/Tokyo）を設定する
- [ ] `@PreUpdate` で `updatedAt` に現在時刻（Asia/Tokyo）を設定する
- [ ] `answers` との OneToMany 関連を定義する（`mappedBy = "question"`）

### Answer エンティティ（`entity/Answer.java`）
- [ ] `@Entity` `@Table(name = "answers")` を付与する
- [ ] `id`：BIGINT PK AUTO_INCREMENT
- [ ] `question`：`@ManyToOne` FK(questions.id)（`@Column(name = "question_id")`）
- [ ] `content`：TEXT NOT NULL
- [ ] `responder`：VARCHAR(50) NOT NULL
- [ ] `createdAt`：DATETIME NOT NULL
- [ ] `updatedAt`：DATETIME NOT NULL
- [ ] `deletedAt`：DATETIME NULL
- [ ] `@PrePersist` / `@PreUpdate` でタイムスタンプを設定する

### インデックス
- [ ] `questions.status` にインデックスを設定する（`@Index`）
- [ ] `questions.created_at` にインデックスを設定する
- [ ] `questions.deleted_at` にインデックスを設定する
- [ ] `answers.deleted_at` にインデックスを設定する

### リポジトリ
- [ ] `QuestionRepository.java`：`JpaRepository<Question, Long>` を継承する
- [ ] `AnswerRepository.java`：`JpaRepository<Answer, Long>` を継承する

### 動作確認
- [ ] アプリ起動時に `questions` / `answers` テーブルが自動生成されることを確認する（`ddl-auto: update`）
- [ ] タイムスタンプが Asia/Tokyo で保存されることを確認する
